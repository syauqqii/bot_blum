### How to use
1. Clone Repository
```
git clone https://github.com/syauqqii/Bot_BLUM
```

3. Open Folder
```
cd Bot_BLUM
```

3. Install Dependencies
```
npm i
```

4. Run application
```
npm start
```

5. Add account
```
Get URL to game, and paste it (with params 'tgWebAppData')
ex: https://telegram.blablabl.io/#tgWebAppData=blablablablablabla
```
